# Pedrungeon (Dragones & Pedrulos) 🐉
Un motor de RPG narrativo que utiliza Inteligencia Artificial Generativa para crear aventuras dinámicas y únicas en cada sesión.

## 💻 Características Técnicas
- **Motor de IA:** Integración con la API de **Google Gemini** para la generación de diálogos y eventos.
- **Backend:** Servidor en Node.js (ESM) con soporte para CORS y ejecución concurrente de cliente/servidor.
- **Middleware:** Gestión de prompts complejos para mantener la coherencia narrativa del juego.

## 🌐 Acceso
Juega ahora en: [https://pedrungeon.ddnsfree.com/](https://pedrungeon.ddnsfree.com/)

## 🛠️ Instalación
1. Añade tu `GEMINI_API_KEY` en el archivo `.env`.
2. `npm install`
3. `npm start`
